# Genomics Figure Pipeline

This repository contains scripts for manuscript figure generation, including genome-wide heatmaps, subtraction heatmaps, loop circos plots, and Venn diagrams.

## Modules

- `genome_wide_heatmap/`: genome-wide heatmap analysis
- `subtraction_heatmap/`: subtraction or differential heatmap analysis
- `loop_circos_plot/`: loop circos visualization
- `venn_diagram/`: Venn diagram generation
